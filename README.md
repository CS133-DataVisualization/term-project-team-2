
# RAM Price Analysis & Prediction

This project analyzes historical RAM pricing data and builds machine learning models to predict RAM prices based on features such as generation, brand, capacity, and form factor.

We perform:

- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine learning model training
- Model evaluation using cross-validation

The models used include:

- Polynomial Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
## Environment Setup

Step 1. Clone the Repository git clone: https://github.com/HardwareDealsCo/ram-deals.git

Step 2. Install required libraries: 
    
    pip install pandas numpy scikit-learn plotly panel hvplot seaborn matplotlib geopy jupyter_bokeh

Main Libraries Used:
- pandas / numpy → data manipulation
- scikit-learn → machine learning models & pipelines
- plotly / matplotlib / seaborn → visualization
- panel / hvplot → interactive dashboard
- geopy → geospatial features

## How to Run Project

Step 1: Launch Jupyter Notebook 

Step 2: Open the Notebook: ```Project_Machine_Learning.ipynb```

Step 3: Run all cells in order from top to bottom

Step 4: Make sure dataset is cloned correctly
